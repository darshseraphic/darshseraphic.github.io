Welcome to the source repository for my professional engineering interface. This repository houses the single-file, high-performance web directory built to catalog my engineering capabilities, technical inventory, structural philosophies, and flagship software builds.



##  Design Paradigm & Architecture

Following the exact design system engineered for **Asrio**, this interface completely strips away commercial web paradigms, heavy tracking libraries, and unneeded visual noise.

* **Typography:** Configured to prioritize the premium **SF Pro Display** / **SF Pro Text** typographic system for absolute tracking precision and structural hierarchy.
* **Visual Grid:** Built upon high-contrast, multi-shade greyscale parameters using sharp geometric containers and custom `0.8px` hairline borders (`#1F1F1F`).
* **Performance Layer:** 100% dependency-free native HTML5 and modern CSS variables. Zero framework overhead, zero layout shift, and immediate local execution.
* **Responsiveness:** Programmed with dynamic viewport calculations to seamlessly scale down across mobile form factors without breaking grid alignments or typographic weighting.
